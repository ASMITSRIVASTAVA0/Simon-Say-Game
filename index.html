Simon Say Game

Press any KeyBoard key to start the game







Rules to Play the Game

The device has four colored buttons, each producing a particular tone when it is pressed or activated by the device. A round in the game consists of the device lighting up one or more buttons in a random order, after which the player must reproduce that order by pressing the buttons.
